# ProjectFinances

## Instruções 
- Executar o npm install
- Criar o .env com a varaivel MONGO_DB (Aqui ficar o seu bando de dados mongo)
- Se for rodar como dev, executar o npm run dev
- Se for rodar como prd, executar npm run start


### Tecnologias usadas
- express
- mongoose
- babel
- date-fns
- eslint
- prettier
